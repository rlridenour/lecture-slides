## Purpose

This repository contains a custom package for [Typst](https://typst.app/), a modern markup-based typesetting system. The package is used to create slides similar to those generated by LaTeX Beamer class with speaker notes to be displayed on a second screen. The package can also create a handout with more detailed notes from the same source file. The sample-lecture directory contains what I currently do in LaTeX that I'm trying to replicate in Typst. The original source file is in Org mode which gets exported to LaTeX and compiled to PDF.

## Typst Basics

Compile a `.typ` file to PDF:

```sh
typst compile slides.typ
```

Watch for changes and recompile automatically:

```sh
typst watch slides.typ
```
