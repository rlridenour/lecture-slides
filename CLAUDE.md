## Purpose

This repository contains lecture slides built with [Typst](https://typst.app/), a modern markup-based typesetting system.

## Typst Basics

Compile a `.typ` file to PDF:

```sh
typst compile slides.typ
```

Watch for changes and recompile automatically:

```sh
typst watch slides.typ
```
